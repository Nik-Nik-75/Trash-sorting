A small game about sorting trash, still very much a work in progress. 
This is my first project using JavaScript, and I’m learning everything from the internet.

I’m doing my Freiwillige Ökologische Jahr, and every FÖJ participant has to create a small project related to the ecology and/or the company where they work. I knew that after this I wanted to become a programmer, and I was also lucky enough to get the opportunity to help develop our company’s new website. I had no experience working with HTML, CSS, and JavaScript, but I started learning and realized that I could create a small game for their website as my project, and at the same time learn JavaScript even better.

Plan:
